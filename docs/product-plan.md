# Plan de producto

## MVP

- Capturar foto de un ticket.
- Extraer conceptos de forma automatica.
- Confirmar el total detectado.
- Seleccionar productos completos por comensal.
- Oscurecer productos ya asignados.
- Ver resumen final por persona.
- Copiar el resumen.

## Fuera del MVP

- Login.
- Historico de tickets.
- Division parcial de un producto entre varias personas.
- Pagos entre usuarios.
- Gestion avanzada de propinas e impuestos.

## Criterio de exito

La app debe permitir resolver una cuenta real en menos de dos minutos, incluso cuando la extraccion automatica falle parcialmente.
